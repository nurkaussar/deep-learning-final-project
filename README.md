## About the Project
The goal of this project is to develop and train a Deep Learning model to classify seven basic human emotions (angry, disgust, fear, happy, sad, surprise, neutral) from 48x48 pixel grayscale facial images. 

The primary dataset used for this task is the **FER-2013** dataset.

## Repository Structure

This repository follows the structure below:

* `data/` — Instructions and scripts for downloading the dataset. **Note:** Raw image files are not uploaded to GitHub due to file size constraints.
* `notebooks/` — Jupyter notebooks containing Exploratory Data Analysis (EDA) and model training experiments.
* `src/` — Python source code for data preprocessing and model architectures.
* `reports/` — The initial project proposal and weekly progress reports.
* `results/` — Saved training curves (loss/accuracy), confusion matrices, and the best model weights.

## How to Get the Data
The dataset for this project is hosted on Kaggle. To run the code locally:
1. Download the dataset archive from the following link: [FER-2013 on Kaggle](https://www.kaggle.com/datasets/msambare/fer2013)
2. Extract the archive into the `data/` directory (ensure that the `train` and `test` folders are placed directly inside).
